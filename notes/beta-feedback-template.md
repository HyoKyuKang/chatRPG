## 친구 5명 모집 가이드

**누구를 부를까:**
- **2명** — 게임 좋아하는 친구 (RPG / 텍스트 게임 / 인디 익숙)
- **2명** — 일반 모바일 유저 (큰 게임만 하던 친구. 우리 톤이 통하는지 봄)
- **1명** — 비평 잘 해주는 친구 (글솜씨 / UX 감 좋은)

**언제 부탁:**
- 5d (Google Play 자산 준비) 시작 시점에 동시 (3~5일 passive 수집).
- 친구가 부담 안 갖게 "1회차만이라도 OK" 강조.

**언제 안 부탁:**
- AdMob 들어간 V1.1 베타 — 광고 경험은 다른 친구 5명 (재방문 X).

## Triage 가이드 (수집 후)

피드백을 두 갈래로:

### V1.0 blocker (출시 전 fix 필수)
- ✅ 재현 가능한 크래시 (앱이 꺼진다)
- ✅ 진행 막히는 버그 (선택지 안 눌림 / 글자 깨짐 / 화면 잘림)
- ✅ 한국어 인코딩 깨짐
- ✅ 시작 시 검은 화면 / 무한 로딩

→ `notes/v1.0-blocker.md` 에 박음. 재현 가능하면 즉시 fix.

### V1.1 backlog (출시 후 무료 업데이트)
- ⏭ "어디서 헷갈렸어요" — UX / 글 톤 catch
- ⏭ "재미 없었어요" — 콘텐츠 / 분량 catch
- ⏭ "이런 거 있으면 좋겠어요" — 새 기능 요청 (combat / 인벤 / 클래스)
- ⏭ 아주 드문 디바이스 호환 (Android 6 이하 / 작은 화면)

→ `notes/v1.1-backlog.md` 에 박음. V1.1 plan 시 prioritize.

## 응답 안 오면

3일 무응답 = 재촉 1번:
> "혹시 시간 나면 한 번만 켜봐줘! 5분 만으로도 좋아."

5일 무응답 = 다른 친구로 대체.

목표: 출시 전 **3+ 친구 install 성공 + 1+ 친구가 한 엔딩 도달**.
