## 트랙 A — Reviewer retroactive QA (네이밍 / 글톤 / 디테일)

> V1.7 디자인 문서의 "Reviewer retroactive QA" 트랙. 한 번에 reviewer subagent 돌려서 일관성 검토.

### A-1. "수상자" 이름 재고 (2026-05-01)

**Found:** 첫 풀 체감 플레이 중. 5지역 reveal ("나는 너 이전의 자다") 의 톤과 비교하면 "수상자" = "수상한 자" 가 너무 직설적, 미스터리 결 약함.

**왜 단순 네이밍이 아닌가:**
- 5지역 reveal 시 정체 = "너 이전의 이방인". 이 컨셉과 호환되어야 함.
- 1~4지역 매 등장 = 점진적 미스터리 빌드업. 이름 자체가 "그가 무엇인가" 의 첫 단서.
- 마왕 / 영웅 풀 / 잔향 시스템 톤과 일관 필요 (e.g., "그림자 사령관" / "잠식된 자" / "잠든 왕" 류와 한 결).

**후보:**
- "흰 옷의 그림자" — dc-resolution 에 이미 묘사됨, 시각적 일관
- "잿빛 그림자" — 5지역 첨탑 / 회색 톤
- "회색의 자" — 영웅 풀 패턴 ("잠식된 자", "잠든 왕") 따름
- "너 이전의 자" — 정체 자체가 이름, reveal 임팩트 강함 (단점: 1~4지역엔 너무 직설)
- "첫 번째 자" — 추상, 신비, 빌드업 가능
- "파수자" / "잿빛 파수자" — watcher 결 (잔향 시스템 watcher 와 결 일치)

**처리 방법:** Reviewer 라운드에서 결정. 이름 후보별 5지역 reveal 임팩트 시뮬레이션.
