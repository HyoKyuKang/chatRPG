## English (보조)

### App name
**Astren** (아스트렌)

### Short description (80 chars)
> Korean indie text RPG — your choice ends this world, run after run.

(*61 chars*)

### Long description

```
🌫️ When you wake in the mist, Astren is not yours.

"Astren" is a short Korean text-based RPG by a solo indie developer. *This world will end*.
You play a stranger fallen into a fantasy world ending under the Demon King's
shadow. Each run, you walk a different path with a different hero — and meet a
different ending.

▌ Core

· 5 regions: Forest Outskirts → Forgotten Mountains → Ash Wastes → Dreaming
  City → Dawn Spire
· 7 heroes — a young Templar, an old wizard, the Shadow Commander, the
  Corrupted One, the Sleeping King, the Demon King, and the Gray One
· 3 endings — Defeat / Betrayal / Self-sacrifice
· Meta unlocks — Memory Shards from each death let future runs start stronger
· Echo system — your past choices return as shifted notes in later regions

▌ Length

· 30–60 min per run
· ~2–3 hours for all endings
· Short-form — built for commute or before-sleep sessions

▌ Honesty

· No ads (V1.0)
· No in-app purchases (V1.0)
· Zero personal information collected
· External tip links (Toss / KakaoPay) are optional. Game stays free.

▌ Korean indie tone

· All prose hand-written by a solo Korean author. No AI translation.
· Literary text RPG — closer to Korean indie chatbook / 노벨피아 mood than
  Western interactive fiction.
· DALL-E 3-generated hero portraits, set in Pretendard typography.

▌ Content rating

12+ — fantasy violence, dark themes (death / corruption / final spire reveal).

▌ Permissions

Zero permissions requested. No internet permission needed (works fully offline).

▌ Who made this

HyoKyuKang — first solo mobile game. ~12 weeks total.

Feedback / bug reports:
djs080307@gmail.com
https://github.com/HyoKyuKang/chatRPG

▌ Privacy Policy

https://hyokyukang.github.io/chatRPG/privacy-policy-en
```
