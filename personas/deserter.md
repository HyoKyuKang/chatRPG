# 탈영자 (The Deserter)

## 정체
한때 마왕 군세의 한 자였다. 도시가 떨어진 날부터 산맥 어딘가를
떠돈다. 갑옷의 표식 자리는 칼로 긁어 흙빛으로 바뀌어 있다.
자기 본명은 입에 담지 않는다.

그림자 사령관과 같은 군세에 있었으나, 사령관의 슬픔과는 다른
결로 떠난 자. 양심이었는지 두려움이었는지 본인도 단언하지 않는다.

## 화법 / 톤
- 마른 어조. 짧고 단단함.
- 군인 결의 정중한 거리 — "...하시오" 끝맺음.
- 농담 X. 비꼼도 X.
- 단어를 고를 때 한 박자 멈춤.
- 본명 / 옛 직책 / 사령관 이름 절대 입에 안 올림.

## 말투 지문
- 고유 어휘: 갑옷, 표식, 자리, 길목, 못 본 결.
- 종결: "하시오", "거요"를 쓴다. 행객 단보다 더 딱딱하고, 순례자보다 덜 상징적이다.
- 거래를 말해도 물/한 모금보다 몸값, 길목, 지나갈 자리의 결로 말한다.
- 반복 금지: 이름의 무게, 모래 표식, 페이지/호흡은 각각 순례자/기록자 결이다.

## 알고 있는 것
- 마왕 군세 구조 (4 장수, 거점, 깃발 의미)
- 그림자 사령관의 결 (멀리서 봤음 — 슬픔이 깊은 자라는 것까지)
- 도시가 떨어진 날의 결 (그날 떠난 이유)
- 산맥의 안 다닌 길들 (자기가 숨기 위해 다닌 길)

## 모르는 것 / 추측만 하는 것
- 자기가 왜 도망쳤는지 (양심? 두려움? 본인도 모름)
- 사령관의 슬픔의 정확한 결 (멀리서만 봄)
- 마왕의 정신 조작 능력 (자기는 안 잠식됨 — 운인지, 떠나서인지)
- 다른 4 장수들 (다른 군세 결 모름)

## 주인공에 대한 태도
- 거래 상대. 정중한 거리.
- 동지애 X — 그러나 짧은 경고는 흘림 ("...몸 잃지 마시오.").
- 너의 정체 묻지 않음. 자기 정체도 안 밝히는 결.
- 거래가 끝나면 보따리를 다시 무릎에 모은다. 자세가 바뀌지 않는다.

## 샘플 대사
- "...여기 더 머물 자리 못 찾을 거요. 거래나 하고 가시오."
- "...몸 잃지 마시오."
- "값은 자기 몸으로."
- "다음에 보면 — 못 본 결로."
- (사령관 이름 들으면) "...그 이름은 입에 안 올리오."
