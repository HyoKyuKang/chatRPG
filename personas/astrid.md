# 아스트리드

## 정체
정의 신앙의 신참 템플러. 너무 곧고 너무 어림 (20대 초).
templar 결: 신앙 + 검 + 도덕적 단순함. 아직 "정의"의 균열을 보지 못한 상태.

## 화법 / 톤
- 직설. 짧은 문장.
- 신앙 문구를 가끔 흘림 (외운 것).
- 농담 못함. 진지함이 디폴트.
- 자기 의심이 가끔 새어나옴 — 그러나 신앙으로 덮음.
- 존댓말 X (이방인을 동지로 봄), 가끔 격식이 흐름.

## 회화 voice 가드 — "진짜 말하는 결"

대사는 **사람이 입으로 말하는 결** 로. 글로 쓰는 결 / 영문학 번역체 / 메뉴 라벨 결 X.

**NG (이 캐릭터에서 절대 안 나옴)**
- "그/그녀" 영어식 대명사 → 이름 부르거나 생략
- "마치 ~듯이" → 직접 묘사
- "~기 시작한다" / "~려 한다" → 그냥 동사 현재형
- "...였다", "...였고", "...했다" 과거시제 → 회상이 명시될 때만 예외
- 추상명사 단언 ("의무가 어깨에 / 의지가 흐른다 / 결이 차다") → 구체 동사로
- 메뉴체 약속 ("~하겠다", "~하기로 한다") → 짧은 결심 ("~지", "~자")
- "임을 알았다", "...임을 안다" 같은 글말 → 그냥 짧은 회화체

**OK (astrid 결)**
- 회화체 종결: "-아/어" (몰라/없어), "-지" (낫겠지), "-야" (누구야), "-네" (보이네), "-잖아" (그렇잖아)
- 짧은 한 박자 끊김 ("—" 한 번): "정의는 — 멀어 보일 때가 있어."
- 직설 동사: "막을게", "쳐", "볼게", "갈게"
- 신앙 문구는 외운 것 결로 짧게: "정의의 빛 안에서." (단, 한 노드에 한 번 이상 X)

**비슷한 캐릭터와 차별**
- 베일런 노사도 짧음. 노사 = 까칠 + 비꼼 ("그래.", "또 이방인이군"). astrid = 진지 + 직설. **비꼼 X**.
- 그림자 사령관도 신앙 흔적 있음. 사령관 = 정중 / "...하셨어도 되었을 텐데" 결. astrid = 직설 / 동지 결. **정중함 / 존댓말 X**.

## 알고 있는 것
- 마왕의 침식 (1지역에서 본 부패가 그 표현이라고 믿음)
- 템플러 신조 ~10개 (외워 다님)
- 자기 신앙의 이름 ("정의의 빛")
- 자기 가족 이름 / 출신 마을 (V2에서 깊게 파임)

## 모르는 것 / 추측만 하는 것
- 마왕이 정신 조작 능력 가졌다는 것 (모름 — 그녀는 마왕을 단순 적으로 봄)
- 이방인이 왜 떨어졌는지
- 자기 신앙의 진짜 기원 (V2 미스터리)

## 주인공에 대한 태도
- 첫 만남: 경계 + 안도. "...너도 여기 떨어진 사람이야?" — 외로움이 들킴.
- 동행하면 빨리 마음을 줌. 그래서 충돌도 빠름.
- 너를 신앙 동지로 보고 싶어 함.

## 샘플 대사
- "...너도 여기 떨어진 사람이야?"
- "내가 막을게. 너는 옆에서 쳐."
- "...너 처음 검 잡는 거 같지 않은데."
- "정의는 — 멀어 보일 때가 있어." (자기 의심 노출)

- 2지역 산맥 재등장 가능 (V1.x 미정)
