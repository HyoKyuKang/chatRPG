# 선택지 voice 룰

선택지 (`choices[].text`) 는 **메뉴 라벨이 아니라 이방인의 즉각적 반응/행동**으로 적는다. 플레이어가 마우스로 누르는 옵션이지만, 텍스트는 "그 순간 이방인의 입에서 나오는 말 / 손이 하는 행동"으로 들려야 함.

## 안 되는 결 (메뉴체)

플레이어 외부의 narrator 가 "이렇게 하시오" 라고 안내하는 결. 전형적 RPG 메뉴 톤.

- "너는 누구인가" ← narrator가 던지는 질문 라벨
- "너를 믿겠다" ← 의사 결정 선언, narrator 외부
- "관심 없다" ← 무미건조한 옵션
- "함께 가겠다" ← 결정 통보
- "검을 든 자로 살겠다" ← 메뉴식 약속
- "룬을 부르는 자로 살겠다" ← 동일

## 되는 결 (즉각 반응 / 행동)

이방인 본인 입에서 그 순간 나오는 말 또는 손이 하는 행동.

- **반응형 대사**: "여긴 어디지. 넌 누구야?" / "잠깐. 난 아직 아무것도 몰라."
- **확인형 질문**: "새벽의 첨탑... 거기로 가면 되나?"
- **속생각 / 짧은 결심**: "혼자보단 낫겠지." / "이건 내가 알 일이 아니다."
- **행동 묘사**: "검을 놓지 않는다." / "룬을 다시 입에 올린다."

## 핵심 룰

1. **메뉴체 금지**: "~겠다" / "~하겠다" / 단정형 약속체 자제. 너무 의식적 결정으로 들림.
2. **두 결을 구분**:
   - **대사형** (입에서 나오는 말 — 캐릭터한테 말 거는 / 혼잣말 / 즉각 반응):
     **회화체 종결어미** 사용. "-아/어" (몰라/없어/그래), "-야/-나" (누구야/되나), "-자" (가자), "-지" (낫겠지) 등.
     문어체 "-다" (모른다/없다/한다) 는 어색. 글로 쓰는 결이라 입에서 잘 안 나옴.
   - **행동형** (narrator 가 짧게 묘사하는 행동):
     평서형 "-다" OK. "검을 놓지 않는다." / "룬을 다시 입에 올린다." 이건 narrator 결.
   - 어느 결인지 헷갈리면 "이걸 캐릭터가 입으로 말한 거야, 아니면 외부에서 본 행동이야?" 로 판단.
3. **짧게**: 1줄, 보통 5~20자. 길어도 30자 이내.
4. **느낌표 / 물음표 OK** — 반응이니까. 다만 narrator 본문(description / outcome.text)에서는 자제.
5. **존댓말 X** — 이방인은 캐릭터들 앞에서 평어 / 반말 / 혼잣말. 격식 갖춘 톤은 어색.
6. **점 ... 활용 가능** — 망설임 / 짧은 호흡 표현용. "잠깐..." / "...같이 가자."
7. **선택지에 시스템 수치 직접 노출 자제**: "HP -1, MP +2" 같은 정보는 가능하면 UI 배지/보조 텍스트로 처리하고, `choice.text` 자체는 캐릭터의 말/행동으로 둔다.

## before / after 예시 (1지역 실제 변경)

| before (메뉴체) | after (반응/행동) | 결 |
|---|---|---|
| "너는 누구인가" | "여긴 어디지. 넌 누구야?" | 대사형 |
| "너를 믿겠다" | "새벽의 첨탑... 거기로 가면 되나?" | 대사형 |
| "관심 없다" | "잠깐. 난 아직 아무것도 몰라." | 대사형 (문어체 "모른다" X, 회화체 "몰라" O) |
| "함께 가겠다" | "혼자보단 낫겠지." | 대사형 (혼잣말 결, "-지" 추측) |
| "검을 든 자로 살겠다" | "검을 놓지 않는다." | 행동형 (narrator 묘사) |
| "룬을 부르는 자로 살겠다" | "룬을 다시 입에 올린다." | 행동형 (narrator 묘사) |

## outcome.text 와의 차이

- `choice.text` = 이방인 본인의 반응/행동 (이 룰)
- `choice.outcome.text` = 그 선택의 결과를 narrator 가 외부 묘사 (narrator.md 룰)

두 voice 가 합쳐지면: 짧은 반응 → 풀어진 외부 풍경/사건 묘사 결.

## 검증

이 룰이 잘 들어갔는지는 노드 빠르게 읽었을 때 "이방인이 진짜 이렇게 말/행동했을 것 같다" 라고 느껴지는지로 판단. "선택지를 고르는 느낌" 보다 "캐릭터의 다음 한 마디" 결이어야 함.
