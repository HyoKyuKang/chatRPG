# 개인정보 처리방침 — 이 세계는 끝날 거야

**시행일:** 2026년 5월 (V1.0 출시 시점)
**최종 갱신:** 2026년 5월

---

## 1. 개요

"**이 세계는 끝날 거야**" (이하 "본 앱") 는 개인 인디 개발자가 만든 한국어 텍스트 기반
롤플레잉 게임입니다. 본 앱은 **사용자의 개인정보를 일체 수집하지 않습니다.**

본 처리방침은 한국 개인정보 보호법(PIPA), EU 일반 개인정보 보호법(GDPR), 그리고
구글 플레이 데이터 안전 정책에 따라 작성되었습니다.

---

## 2. 수집하는 정보

본 앱은 **어떤 개인정보도 수집·저장·전송하지 않습니다.**

구체적으로:

| 항목 | V1.0 |
|---|---|
| 사용자 식별 정보 (이름 / 이메일 / 전화번호) | 수집 안 함 |
| 디바이스 식별자 (IMEI / 광고 ID) | 수집 안 함 |
| 위치 정보 | 수집 안 함 |
| 사용 통계 / 분석 (Firebase Analytics 등) | 사용 안 함 |
| 광고 SDK (AdMob 등) | 사용 안 함 |
| 크래시 리포트 (자동 전송) | 사용 안 함 |
| 카메라 / 마이크 / 연락처 / 사진 접근 | 요구 안 함 |

게임 진행 상황 (출정 횟수 / 기억의 조각 / 도달한 엔딩 등) 은 **사용자 디바이스의
브라우저 로컬 저장소(localStorage)에만 저장되며, 외부로 전송되지 않습니다.** 앱
삭제 시 함께 사라집니다.

---

## 3. 제3자 제공

수집하는 정보가 없으므로 **제3자에게 제공하는 정보도 없습니다.**

본 앱은 다음과 같은 제3자 서비스를 **사용하지 않습니다**:

- Google Analytics / Firebase Analytics
- Google AdMob (광고)
- Facebook SDK
- 외부 인증 서비스 (Google / Apple / Kakao 로그인)
- 클라우드 백엔드 (서버에 저장되는 데이터 X)

---

## 4. 외부 링크

본 앱은 메뉴에 **외부 후원 링크 (Toss 또는 카카오페이)** 를 포함할 수 있습니다.
사용자가 이 링크를 탭하면 외부 결제 페이지로 이동합니다. **외부 페이지에서의
개인정보 수집 / 처리는 해당 서비스의 약관 / 정책을 따릅니다.** 본 앱은 외부
페이지의 활동에 책임지지 않습니다.

후원은 **앱 내 결제(IAP)가 아닌 외부 결제** 이며, 사용자 디바이스에서 실행되는
브라우저로 이동하여 처리됩니다.

## 5. 어린이 개인정보

본 앱은 **만 13세 미만 어린이의 개인정보를 의도적으로 수집하지 않습니다.**

본 앱은 어떤 개인정보도 수집하지 않으므로 어린이를 포함한 모든 사용자에게 안전합니다.

콘텐츠 등급은 IARC 기준 **12세 이상** 입니다 (판타지 폭력 / 어두운 주제 묘사 포함).

---

## 6. 데이터 보관 / 삭제

수집하는 데이터가 없으므로 보관 / 삭제 처리 절차가 없습니다.

게임 진행 상황 (localStorage) 을 삭제하려면:
- 앱 내 설정에서 "초기화" 버튼 (DEV 빌드만 노출됨)
- 또는 Android 설정 → 앱 → 이 세계는 끝날 거야 → 저장소 → 데이터 삭제
- 또는 앱 자체 삭제

## 7. 사용자 권리

본 앱이 수집하는 개인정보가 없으므로 GDPR / PIPA 가 부여하는 다음 권리는 자동으로
충족됩니다:

- 열람권: 수집된 데이터가 없으므로 N/A
- 정정 / 삭제권: localStorage 자체를 사용자가 직접 삭제 가능 (위 6항)
- 처리 정지권: 수집을 안 하므로 정지할 처리 없음
- 데이터 이동권: 외부에 전송되는 데이터 없음
