# Privacy Policy — This World Will End (이 세계는 끝날 거야)

**Effective:** May 2026 (V1.0 release)
**Last updated:** May 2026

---

## 1. Overview

"**This World Will End**" (Korean: 이 세계는 끝날 거야, hereinafter "the App") is a
Korean text-based role-playing game made by an independent solo developer. **The App
collects no personal information from users.**

This policy is written to comply with Korea's Personal Information Protection Act
(PIPA), the EU General Data Protection Regulation (GDPR), and Google Play's Data
Safety policy.

---

## 2. Information Collected

The App **does not collect, store, or transmit any personal information.**

Specifically:

| Category | V1.0 |
|---|---|
| Identifiers (name / email / phone) | Not collected |
| Device identifiers (IMEI / advertising ID) | Not collected |
| Location data | Not collected |
| Usage / analytics (Firebase Analytics, etc.) | Not used |
| Advertising SDK (AdMob, etc.) | Not used |
| Automatic crash reporting | Not used |
| Camera / microphone / contacts / photos | Not requested |

Game progress (run count / memory shards / endings reached) is **stored only in
localStorage on the user's device and never transmitted externally.** It is removed
when the app is uninstalled.

---

## 3. Third-Party Sharing

Since no information is collected, **nothing is shared with third parties.**

The App does **not** use:

- Google Analytics / Firebase Analytics
- Google AdMob (ads)
- Facebook SDK
- External authentication (Google / Apple / Kakao login)
- Cloud backend (no server-side data)

---

## 4. External Links

The App may include **external donation links (Toss or KakaoPay)** in its menu.
Tapping these opens an external payment page. **Personal information collection /
processing on external pages is governed by those services' terms and policies.**
The App is not responsible for activity on external pages.

Donations are **external payments, not in-app purchases (IAP)**, processed via the
device's browser.

---

## 5. Children's Privacy

The App **does not knowingly collect personal information from children under 13.**

Since the App collects no personal information at all, it is safe for all users
including children.

Content rating per IARC: **12+** (fantasy violence / dark themes).

---

## 6. Data Retention / Deletion

Since no data is collected, there are no retention or deletion procedures.

To delete game progress (localStorage):
- In-app "reset" button (DEV builds only)
- Or Android Settings → Apps → This World Will End → Storage → Clear data
- Or uninstall the app

---

## 7. User Rights

Since the App collects no personal information, the following GDPR / PIPA rights are
automatically satisfied:

- Right to access: nothing collected, so N/A
- Right to rectify / erase: localStorage can be deleted directly by the user (see §6)
- Right to restrict processing: no processing occurs
- Right to data portability: no data is transmitted externally
